# Ridge and Lasso Regression

A simple Python project that demonstrates **Ridge Regression** and **Lasso Regression** using Scikit-learn. The project shows how regularization helps improve model performance and reduce overfitting.

## 📌 Features

* Load and preprocess data
* Train Ridge and Lasso regression models
* Compare model performance
* Evaluate using common regression metrics

## 🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/Vedline2547/Ridge-Lasso-Regression.git
```

2. Install the required packages:

```bash
pip install -r requirements.txt
```

3. Run the project:

```bash
python main.py
```
